# xTracker — Интимный трекер для пар

## Как установить

### Что нужно
- Mac с установленным Xcode
- iPhone
- Apple ID (бесплатный)

### Шаги

**1. Скачай проект**
- Зайди на github.com/yasterbog/xTracker
- Нажми Code → Download ZIP
- Распакуй архив

**2. Открой в Xcode**
- Открой файл xTracker.xcodeproj

**3. Настрой подпись**
- Нажми на xTracker слева (синяя иконка)
- Вкладка Signing & Capabilities
- В поле Team выбери свой Apple ID
- Если Apple ID нет — добавь в Xcode → Settings → Accounts

**4. Подключи iPhone**
- Подключи кабелем к Mac
- На телефоне нажми Доверять
- Выбери свой iPhone в Xcode вверху

**5. Установи приложение**
- Нажми кнопку ▶️ или Cmd+R
- Подожди 2-3 минуты

**6. Разреши запуск на телефоне**
- Настройки → Основные → VPN и управление устройством
- Найди свой Apple ID → Доверять

**7. Подключись к паре**
- Открой приложение
- Зайди в Настройки
- Скопируй свой код и отправь партнёру
- Партнёр вводит твой код в своих Настройках

### Важно
- Приложение работает 7 дней, потом нужно переустановить (повторить шаг 5)
- Нужен Mac для установки
- Все данные хранятся в облаке и не пропадают при переустановке
